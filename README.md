# Elian Beak — Website

Modern Astro rewrite of elianbeak.de. Static site, Lighthouse 90–100 target.

**Stack:** Astro 4 · TailwindCSS · TypeScript · Vercel

---

## Local Development

```bash
npm install
npm run dev
# → http://localhost:4321
```

## Build

```bash
npm run build
npm run preview  # preview production build locally
```

---

## Deployment (Vercel)

### First deploy

1. Push to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/elianbeak/elianbeak-website.git
   git push -u origin main
   ```

2. Go to [vercel.com](https://vercel.com) → New Project → Import from GitHub

3. Vercel auto-detects Astro. No extra config needed — `vercel.json` is included.

4. Set custom domain: `elianbeak.de` in Vercel project settings → Domains

### Auto-deploy

Every push to `main` triggers a new Vercel deployment automatically.

## Contact Form

The contact form uses [Formspree](https://formspree.io).

1. Create a free Formspree account
2. Create a new form → copy the form ID (e.g. `xabcdefg`)
3. Update the `action` URL in `src/components/ContactForm.astro`:
   ```
   action="https://formspree.io/f/xabcdefg"
   ```

## Performance Notes

- All images: `loading="lazy"` except Hero (`fetchpriority="high"`)
- No JS frameworks — pure Astro + vanilla JS
- IntersectionObserver fade-ins (opacity + transform only — no CLS)
- CSS-only masonry via `columns` (no JS layout thrashing)
- Sitemap auto-generated by `@astrojs/sitemap`
- `robots.txt` in `/public`
